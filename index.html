Drawing HTML

This chapter covers the core of the d3 algorithm. For your prep create a page to explain how some of the pieces work. In particular, explain what the select, append, text,style commands do. People wishing to do more should demonstrate the full effect by using d3 to generate some html (doesn't have to be very complex).

.select()

The select method takes a CSS selector, for example "body", and it will return a reference to the first element that matches the selecter inputted.

.append()

The append method creates a new specified element that is inputted and appends it to the end, but just inside, the selected element that was chosen prior.

.text()

The text method takes a string and adds it to the selected element.

.style()

The style method allows the user to set HTML attributes and CSS properties on selections.

Generating HTML

The code below uses the methods discussed previously in order to create HTML code with the use of JavaScript. The second line incorporates the style method in order to change the color of the text. The output is found below the lines of code.

var something = d3.select("body").append("p").text("Hello World!");
var variable = d3.select("body").append("p").text("something here!").style("color","white");
